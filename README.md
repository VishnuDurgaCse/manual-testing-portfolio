Manual Testing Portfolio covering Flipkart, Zepto, GFG — 60+ test cases, bug reports, UX observations documented in Jira and Excel

## 👩‍💻 About This Project

This repository contains my complete Manual Testing Portfolio covering
real-world applications — Flipkart, Zepto, and GeeksForGeeks.
All test cases, bug reports, and UX observations are documented
professionally using Jira and Excel.
